# GSH Account

#### Version 1.0.0

https://glosyn.github.io/gsh-account/


## Start

To start a project locally, you need to download it

`git clone https://github.com/GloSyn/gsh-account.git`


## Development

To run dev version

`npm run dev`


## Production

To run the production version

`npm run build`


## Deploy

To launch the production version

`npm run deploy`
